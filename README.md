GDG Chatbot is a simple AI-powered assistant built for the GDG Batna community.  
It helps users get quick and clear information about GDG Batna, its events, programs, and how to join.

The project uses:
- React + Vite for the frontend
- Node.js + Express for the backend
- Google Gemini API for AI responses

This chatbot is designed to be lightweight, practical, and community-focused.
